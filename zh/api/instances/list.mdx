---

List Instances 端点返回与你账户关联的每个 GPU 实例。你可以按当前状态或按区域过滤结果，并使用 `limit` 和 `offset` 对大规模实例集合进行分页。这是随时获取正在运行、已停止或待处理基础设施全局视图的主要方式。

## 端点

```
GET https://api.gpucloud.com/v1/instances
```

## 查询参数

<ParamField query="status" type="string">
  按当前状态过滤实例。可接受的值为 `pending`、`running`、`stopping` 和 `terminated`。省略时，返回所有状态的实例。
</ParamField>

<ParamField query="region" type="string">
  按区域标识过滤实例（例如 `us-east-1`、`eu-west-2`）。省略时，返回所有区域的实例。
</ParamField>

<ParamField query="limit" type="integer" default="20">
  单次响应中返回的最大实例数。可接受的范围为 `1`–`100`。
</ParamField>

<ParamField query="offset" type="integer" default="0">
  在返回结果前跳过的实例数量。与 `limit` 结合使用可对完整列表进行分页。
</ParamField>

## 响应字段

<ResponseField name="instances" type="array">
  匹配你查询的实例对象数组。当没有匹配的实例时，返回空数组。

  <Expandable title="Instance object">
    <ResponseField name="id" type="string">
      实例的唯一标识符（例如 `inst_abc123`）。
    </ResponseField>

    <ResponseField name="name" type="string">
      你在创建时为实例指定的人类可读名称。
    </ResponseField>

    <ResponseField name="status" type="string">
      实例当前的生命周期状态：`pending`、`running`、`stopping` 或 `terminated`。
    </ResponseField>

    <ResponseField name="gpu_type" type="string">
      为实例配置的 GPU 硬件类型（例如 `h100-80gb-sxm`）。
    </ResponseField>

    <ResponseField name="gpu_count" type="integer">
      挂载到实例的 GPU 数量。
    </ResponseField>

    <ResponseField name="region" type="string">
      实例部署所在的区域标识。
    </ResponseField>

    <ResponseField name="ip_address" type="string">
      实例的公网 IPv4 地址。当实例仍处于 `pending` 状态时，此字段为 `null`。
    </ResponseField>

    <ResponseField name="disk_image" type="string">
      用于启动实例的磁盘镜像（例如 `pytorch-2.1-cuda-12`）。
    </ResponseField>

    <ResponseField name="disk_size_gb" type="integer">
      根磁盘大小（以 GB 为单位）。
    </ResponseField>

    <ResponseField name="ssh_key_id" type="string">
      获授权访问实例的 SSH 密钥 ID。
    </ResponseField>

    <ResponseField name="created_at" type="string">
      记录实例创建时间的 ISO 8601 时间戳。
    </ResponseField>

    <ResponseField name="hourly_cost_usd" type="number">
      实例当前的每小时费用（以美元计）。
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="integer">
  匹配你过滤条件的实例总数，与 `limit` 和 `offset` 无关。可用于计算页数。
</ResponseField>

## 请求示例

```bash
curl https://api.gpucloud.com/v1/instances \
  -H "Authorization: Bearer <api-key>" \
  -G \
  --data-urlencode "status=running" \
  --data-urlencode "region=us-east-1" \
  --data-urlencode "limit=20" \
  --data-urlencode "offset=0"
```
