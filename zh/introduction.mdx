GPU Cloud 是一个面向开发者、研究人员和团队的高性能计算平台，为那些需要强大 GPU 基础设施但又不想承担物理硬件管理开销的用户量身打造。无论你是在训练大型语言模型、运行分布式推理流水线，还是加速科学模拟，GPU Cloud 都能让你即时使用顶级 GPU —— 包括 A100、H100 和 RTX 4090 —— 并配备持久化存储、灵活的网络以及对开发者友好的 API。几秒钟即可启动一台完全配置好的实例，只为你实际使用的部分付费，并可以随工作负载需求从单节点扩展到完整集群。

## 你可以做什么

<CardGroup cols={3}>
  <Card title="按需 GPU 实例" icon="microchip">
    数秒内即可配置 A100、H100、RTX 4090 及更多型号。选择最适合你工作负载的 GPU 类型、区域和磁盘镜像。
  </Card>
  <Card title="持久化云存储" icon="hard-drive">
    为实例挂载高速 NVMe 或网络卷，数据在重启和重新部署之间持续保留。
  </Card>
  <Card title="灵活的访问方式" icon="terminal">
    通过 SSH、基于浏览器的终端、Jupyter Notebook 或 VS Code 连接任何实例 —— 使用最适合你工作流程的方式。
  </Card>
  <Card title="预构建磁盘镜像" icon="box">

  </Card>
  <Card title="REST API 与 CLI" icon="rectangle-terminal">
