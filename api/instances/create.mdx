---
title: "Create a New GPU Instance on GPU Cloud"
sidebarTitle: "Create Instance"
description: "Provision a new GPU instance by specifying your desired GPU type, region, disk image, and SSH key. The instance starts booting immediately after creation."
---

Creating an instance provisions a dedicated GPU server in the region of your choice and begins the boot sequence immediately. You must supply a GPU type, region, base disk image, and the ID of an SSH key already registered in your account. Once the instance transitions from `pending` to `running` — typically within a few minutes — you can connect over SSH using the `ip_address` returned in the response.

## Endpoint

```
POST https://api.gpucloud.com/v1/instances
```

## Request Body

<ParamField body="gpu_type" type="string" required>
  The GPU hardware type to provision. Use the [GPU Types](/concepts/gpu-types) reference to list all available options (e.g., `h100-80gb-sxm`, `a100-40gb-pcie`).
</ParamField>

<ParamField body="region" type="string" required>
  The region slug in which to launch the instance (e.g., `us-east-1`, `eu-west-2`). GPU availability varies by region.
</ParamField>

<ParamField body="disk_image" type="string" required>
  The identifier of the disk image used to boot the instance (e.g., `pytorch-2.1-cuda-12`). Use the [Disk Images](/concepts/instances) reference to list all available images.
</ParamField>

<ParamField body="ssh_key_id" type="string" required>
  The ID of the SSH public key to authorize on the instance. The key must already be registered in your account. Use the [SSH Keys](/api/ssh-keys) endpoint to manage your keys.
</ParamField>

<ParamField body="name" type="string">
  A human-readable label for the instance. If omitted, GPU Cloud auto-generates a name. Maximum 64 characters; only alphanumerics, hyphens, and underscores are allowed.
</ParamField>

<ParamField body="gpu_count" type="integer" default="1">
  The number of GPUs to attach to the instance. Must be a positive integer supported by the selected `gpu_type`. Defaults to `1`.
</ParamField>

<ParamField body="disk_size_gb" type="integer" default="50">
  The size of the root disk in gigabytes. Minimum value is `50`. Larger disks incur additional storage costs.
</ParamField>

## Response Fields

A successful request returns `201 Created` with the full instance object in the response body.

<ResponseField name="id" type="string">
  The unique identifier assigned to the new instance.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the instance, either as supplied in the request or auto-generated.
</ResponseField>

<ResponseField name="status" type="string">
  The initial status of the instance, always `pending` immediately after creation.
</ResponseField>

<ResponseField name="gpu_type" type="string">
  The GPU hardware type provisioned for the instance.
</ResponseField>

<ResponseField name="gpu_count" type="integer">
  The number of GPUs attached to the instance.
</ResponseField>

<ResponseField name="region" type="string">
  The region where the instance was launched.
</ResponseField>

<ResponseField name="ip_address" type="string">
  The public IPv4 address of the instance. Returns `null` while the instance is still `pending`.
</ResponseField>

<ResponseField name="disk_image" type="string">
  The disk image used to boot the instance.
</ResponseField>

<ResponseField name="disk_size_gb" type="integer">
  The provisioned root disk size in gigabytes.
</ResponseField>

<ResponseField name="ssh_key_id" type="string">
  The ID of the SSH key authorized on the instance.
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the instance was created.
</ResponseField>

<ResponseField name="hourly_cost_usd" type="number">
  The hourly cost of the instance in US dollars. Billing starts as soon as the instance is created.
</ResponseField>

## Example Request

```bash
curl https://api.gpucloud.com/v1/instances \
  -X POST \
  -H "Authorization: Bearer <api-key>" \
  -H "Content-Type: application/json" \
  -d '{
    "gpu_type": "h100-80gb-sxm",
    "region": "us-east-1",
    "disk_image": "pytorch-2.1-cuda-12",
    "ssh_key_id": "key_xyz789",
    "name": "my-training-run",
    "gpu_count": 1,
    "disk_size_gb": 100
  }'
```

```json
{
  "id": "inst_abc123",
  "name": "my-training-run",
  "status": "pending",
  "gpu_type": "h100-80gb-sxm",

  "region": "us-east-1",
  "ip_address": null,
  "disk_image": "pytorch-2.1-cuda-12",
  "disk_size_gb": 100,
  "ssh_key_id": "key_xyz789",
  "created_at": "2024-01-15T10:30:00Z",
  "hourly_cost_usd": 3.50
}
```
